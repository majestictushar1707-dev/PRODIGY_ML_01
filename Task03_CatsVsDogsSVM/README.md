# Task 03 - Cat vs Dog Image Classification using SVM

## Objective
Implement a Support Vector Machine (SVM) model to classify images of cats and dogs.

## Dataset
Dogs vs Cats (Kaggle)

## Technologies Used
- Python
- OpenCV
- NumPy
- Scikit-learn

## Steps Performed
1. Loaded cat and dog images.
2. Resized images to 64×64 pixels.
3. Converted images into feature vectors.
4. Split the dataset into training and testing sets.
5. Trained an SVM classifier.
6. Evaluated the model using accuracy.

## Result
Model Accuracy: 67.5%
