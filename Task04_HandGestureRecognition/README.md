# Task 04 - Hand Gesture Recognition using SVM

## Objective
Develop a Hand Gesture Recognition model that can identify and classify different hand gestures from image data.

## Dataset
LeapGestRecog (Kaggle)

## Technologies Used
- Python
- OpenCV
- NumPy
- Scikit-learn

## Steps Performed
1. Loaded the LeapGestRecog dataset.
2. Converted images to grayscale.
3. Resized images to 64 × 64 pixels.
4. Converted images into feature vectors.
5. Split the dataset into training and testing sets.
6. Trained a Support Vector Machine (SVM) classifier.
7. Evaluated the model using accuracy.

## Result
Model Accuracy: 99.85%
