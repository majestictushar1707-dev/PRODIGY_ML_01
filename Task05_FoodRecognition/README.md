# Task 05 - Food Recognition using SVM

## Objective
Develop a machine learning model to recognize food items from images and estimate calorie information.

## Dataset
Food-101 (Kaggle)

## Technologies Used
- Python
- OpenCV
- NumPy
- Scikit-learn

## Steps Performed
1. Loaded a subset of the Food-101 dataset.
2. Resized images to 64 × 64 pixels.
3. Converted images into feature vectors.
4. Split the dataset into training and testing sets.
5. Trained a Support Vector Machine (SVM) classifier.
6. Evaluated the model using accuracy.

## Result
Model Accuracy: 35%

> Note: A subset of the Food-101 dataset was used for faster training. Food recognition generally achieves better performance with deep learning models such as CNNs.
